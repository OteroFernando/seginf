Technical-Specifications

FLAM Time-stamping is an HTTP service and time stamp requests are submitted as HTTP POST requests.

 
Address of service https://0.0.0.0:12346/tsr
Service certificate FLAM TIMESTAMPING AUTHORITY
Time-stamping service policy OID 0.4.0.2023.1.1
Supported hash algorithms for time stamp request SHA1, SHA256, SHA384, SHA512
Time stamp signing key parameter 2048bit RSA
Time stamp signing algorithm SHA256withRSA
Time-stamping service policies supported in the request 0.4.0.2023.1.1
Time-stamping service extensions supported in the request None
Responses to correct requests include the following
  • status info – Operation Okay,
  • signing hash algorithm,
  • time stamp policy OID,
  • time-stamping time,
  • time-stamping service certificate (without a full certificate chain),
  • nonce value,
Responses to incorrect time stamp requests include the following
  • status info – Rejected,
  • cause of error
Response, if not a time stamp request HTTP status code: 400 – Bad Request
